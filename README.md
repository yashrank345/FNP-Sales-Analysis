# Sales Analysis Dashboard (Excel)

This project showcases an **interactive Sales Analysis Dashboard** built using **Microsoft Excel**, leveraging **PivotTables**, **Power Pivot**, and **Slicers** for dynamic filtering.

---

## Features

### Key Metrics
- Total Orders  
- Total Revenue  
- Average Customer Spending  

### Revenue Breakdown
- By Occasions  
- By Categories  
- By Month  
- By Hour (Order Time)  

### Top Insights
- Top 5 Products by Revenue  

### Interactive Filters
- Delivery Date  
- Order Date  
- Occasion  

---

## Tools Used
- Excel PivotTables  
- Power Pivot Data Model  
- Pivot Charts  
- Excel Slicers

## How It Works

1. Data is imported and cleaned inside Excel.  
2. Power Pivot is used to build the data model and relationships.  
3. PivotTables are created for summaries and KPIs.  
4. PivotCharts visualize key insights.  
5. Slicers enable dynamic, real-time filtering across the dashboard.
